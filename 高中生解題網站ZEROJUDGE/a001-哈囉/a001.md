##  題目說明  

輸入一行字串，請輸出 `"hello, "` 加上輸入的名字。  
本題會不斷地輸入多筆資料，直到輸入結束。

---

##  解題思路  

這是一題 **基本輸入輸出練習**，考察的是如何使用 C++ 進行連續輸入與對應輸出。解題重點如下：

1. 使用 `cin >> a` 持續讀入直到檔案結束（EOF）。
2. 每讀入一筆資料，輸出 `hello, ` 加上名字。
3. 使用 `'\n'` 而非 `endl`，可避免 flush，提升效能。


---

##  程式碼實作  

```cpp
#include <iostream>
using namespace std;

int main()
{
    string a;
    while(cin >> a)
    {
        cout << "hello, " << a << '\n';
    }
}
