## 題目說明  

給定一個單向鏈結串列，對每個節點找出其後面第一個比它大的節點值，若不存在則為 0，回傳整數陣列。

---

## 解題思路  

1. 先遍歷鏈結串列取得節點數量 `count`，以便分配結果陣列空間。  
2. 使用雙重迴圈：  
   - 外層指向當前節點。  
   - 內層尋找當前節點之後第一個值較大的節點。  
3. 找到後記錄該節點值，沒找到則為 0（初始為 0 已 calloc）。  
4. 回傳結果陣列。
